## 三、架构方案（解决方案）

### 0、方案概述

+ 1.通用授权方式
    + 最常用的授权方式，标准OAuth2授权协议
+ 2.隐式授权方式
    * 又叫JS授权，如果你是单纯的页面应用（无服务器端），这是你的选择
+ 3.企业级授权方式
    * 最直接的授权方式，用户名密码直接授权，由于这种授权方式会暴露用户密码给第三方
